# iOS Starter Structure

This folder contains a source layout for the iOS app. Because the current environment does not include full Xcode tooling, these files are initialized as a clean source scaffold that you can drop into an Xcode iOS App target.

## Next Steps In Xcode

1. Create an iOS App project named `MindMargin`
2. Add these folders as groups
3. Paste the contents into the app target
4. Add the Supabase Swift package:
   - `https://github.com/supabase/supabase-swift`
5. Add capabilities:
   - `HealthKit`
   - `Background Modes`
6. Add the required usage description keys in `Info.plist`

## Suggested Deployment Settings

- iOS 17+
- SwiftUI lifecycle
- `SwiftData` enabled
- Supabase URL and anon key stored in app config
